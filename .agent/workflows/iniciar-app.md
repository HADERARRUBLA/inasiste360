---
description: Cómo iniciar la aplicación Asiste360
---
Para iniciar la aplicación en entorno de desarrollo, sigue estos pasos:

1. Abre una terminal en la carpeta raíz del proyecto.
2. Navega a la carpeta del frontend:
   ```powershell
   cd frontend
   ```
3. Instala las dependencias (solo si es la primera vez o si se han agregado nuevas):
   ```powershell
   npm install
   ```
4. Inicia el servidor de desarrollo de Vite:
   ```powershell
   npm run dev
   ```
5. Abre tu navegador en la dirección que indique la terminal (usualmente `http://localhost:5173`).
